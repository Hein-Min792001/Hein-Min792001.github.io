Side Project · Spring Boot + Angular

Payment Reconciliation Engine

A digital wallet system — think a mini PayPal built from scratch

View on GitHub

A full-featured digital wallet — deposit, transfer, and refund — built to tackle the hard parts of financial software that most tutorials skip.

Pessimistic locking prevents concurrent overdrafts. A Redis idempotency layer stops double-charges on duplicate requests. The Outbox Pattern guarantees RabbitMQ notifications survive broker failures.

Key Features

  • Wallet creation, deposit, transfer & refund
  • Duplicate-payment protection via Redis idempotency
  • Async notifications with RabbitMQ + Outbox Pattern
  • Paginated transaction history

Engineering Highlights

  • Balances stored as Long (cents) — zero floating point risk
  • Double-entry ledger: every transfer = 2 immutable rows
  • DB-level CHECK constraint as a last safety net

Enterprise Warehouse Management System

Full-stack WMS — from purchase order to last-mile delivery

View on GitHub

A production-grade warehouse system covering the full supply chain — from purchase orders and goods receiving all the way to picking, packing, and shipping. Built to scale with 100,000+ products and millions of stock records.

Stock updates are protected with optimistic locking to prevent race conditions. Every action is recorded in an immutable audit trail. 7 fine-grained roles control exactly what each user can see and do.

Key Modules

  • Multi-warehouse inventory management
  • Purchase Orders → Goods Receipt workflow
  • Sales Orders → Picking → Packing → Shipping
  • Real-time stock balances with low-stock alerts

Engineering Highlights

  • JWT auth + refresh tokens + account lockout
  • 34 tables via Flyway versioned migrations
  • Scheduled jobs for alerts & token cleanup
  • Full audit log on every system action
